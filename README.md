## Cómo ejecutarlo?

1. Escribir los datos para operar en un fichero.

2. Ejecutar el codigo ubicado en `App_main/target/classes` con el siguiente comando:
   
   ```bash
   java -jar <Projecto.jar> <ruta>
   Ejemplo:
   java -jar App_main-1.0-SNAPSHOT.jar ./datos.txt
   ```
